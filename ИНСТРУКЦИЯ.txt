## ШАГ 1: УСТАНОВКА ЗАВИСИМОСТЕЙ

1.1) Убедись, что у тебя установлен Python 3.6+ (проверь версию):
     - Открой PowerShell
     - Введи: python --version
     - Если Python не установлен → скачай отсюда: https://www.python.org/downloads/

1.2) Установи OpenCV (главная библиотека для компьютерного зрения):
     - Открой терминал/cmd
     - Введи: pip install opencv-python
     - Жди установки (занимает 1-2 минуты)

1.3) Убедись, что NumPy установлен (обычно устанавливается с OpenCV):
     - Введи: pip install numpy
     

## ШАГ 2: ПОДГОТОВКА ВИДЕОФАЙЛА

2.1) Найди видео с людьми, входящими/выходящими через дверь или проход
     - Формат: MP4, AVI, MOV (любой распространённый)
     - Разрешение: от 320x240 до 1920x1080 (лучше 720x480 или выше)

2.2) Переименуй видео в "video.mp4" (ИЛИ отредактируй путь в коде)
     - Скопируй видео в папку, где лежит people_counter.py
       (В твоём случае: C:\Users\bbebr\Downloads\Telegram Desktop\Новая папка (17))

2.3) Если видео называется иначе:
     - Открой "people_counter.py" в текстовом редакторе (или VS Code)
     - Найди строку: VIDEO_PATH = 'video.mp4'
     - Замени на: VIDEO_PATH = 'your_video.mp4' (если видео называется иначе)
     - Сохрани файл (Ctrl+S)


## ШАГ 3: ЗАПУСК ПРОГРАММЫ

3.1) Открой Command Prompt (cmd) или PowerShell

3.2) Перейди в папку со скриптом:
     cd "C:\Users\bbebr\Downloads\Telegram Desktop\Новая папка (17)"

3.3) Запусти программу:
     python people_counter.py

3.4) Откроется окно с видео — дальше следуй инструкциям в окне


## ШАГ 4: РАБОТА С ПРОГРАММОЙ (во время воспроизведения)

4.1) УСТАНОВКА ЛИНИИ ПЕРЕСЕЧЕНИЯ (очень важно!):
     
     Способ 1 - МЫШКОЙ (рекомендуется):
     - Кликни и перетащи мышь на видео
     - От начальной точки к конечной точке
     - При отпускании мыши → линия установится автоматически
     - Если линия горизонтальная → считает "вход" (сверху вниз) и "выход" (снизу вверх)
     - Если линия вертикальная → считает "вход" (слева направо) и "выход" (справа налево)

     Способ 2 - КЛАВИШИ:
     - Нажми "H" → горизонтальная линия (посередине экрана)
     - Нажми "V" → вертикальная линия (посередине экрана)
     - Нажми "C" → сброс линии (вернуть в центр, можно заново нарисовать)

4.2) ОТОБРАЖАЕМАЯ ИНФОРМАЦИЯ:
     - Зелёные боксы → обнаруженные люди
     - Красные точки → центры объектов (центроиды)
     - Синяя линия → линия для подсчёта
     - "Entered" (зелёный текст) → сколько вошло
     - "Exited" (красный текст) → сколько вышло
     - "Line: horizontal/vertical" → текущая ориентация линии

4.3) УПРАВЛЕНИЕ:
     - Q → завершить программу и закрыть окно
     - H → переключиться на горизонтальную линию
     - V → переключиться на вертикальную линию
     - C → сбросить линию (вернуть по центру)
     - Мышь (drag) → нарисовать новую линию вручную


## ШАГ 5: НАСТРОЙКА ПАРАМЕТРОВ (если всё считает неправильно)

Если программа считает слишком много или мало людей, отредактируй параметры:

5.1) Открой people_counter.py в текстовом редакторе (VS Code, Notepad++, etc.)

5.2) Найди параметры в начале файла:

     MIN_AREA = 1500
     → Если видео в узком коридоре (люди далеко): увеличь на 2000-3000
     → Если видео крупный план (люди близко): может помочь уменьшение

     MAX_ASPECT_RATIO = 4.5
     → Если считаются палки/тени: уменьши на 3.0
     → Если пропускаются полные люди: увеличь на 5-6

     MORPH_KERNEL_SIZE = 5
     → Если много шума: увеличь на 7-9
     → Если люди "разваливаются": уменьши на 3

     В функции merge_boxes(boxes, max_gap=50):
     → Если считаются отдельно руки/ноги: увеличь max_gap на 80-100
     → Если сливаются два рядом стоящих человека: уменьши на 30-40

5.3) Сохрани изменения (Ctrl+S) и запусти заново программу


## ШАГ 6: РЕЗУЛЬТАТЫ

После обработки всего видео (или нажатия Q):
- Программа закроется
- В консоли выведет финальную статистику
- Финальные счётчики показаны в окне до закрытия
