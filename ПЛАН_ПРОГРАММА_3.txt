СОСТАВ РАБОТ (по документу заказчика + уточнения)
══════════════════════════════════════════════════════════════

Блок 1. Распознавание объекта по силуэту
   — выделение силуэта, центр тяжести, нормализация
   — веер лучей из центра, полярные координаты
   — сравнение дескриптора с базой данных
   — запрос «Что это? Кто это?» при неизвестном объекте

Блок 2. Описание картинки
   — выделение объектов на изображении
   — формирование текстового описания сцены

Блок 3. Построение картинки по тексту
   — анализ простого предложения
   — размещение объектов слоями

Блок 4. ВК-словарь (реконструкционный словарь русского языка)
   — ВНИМАНИЕ: это НЕ обычный словарь, а реконструкция русских
     слов через древние образы-слоги (ВК-язык)
   — Формат: слово -> разбивка на слоги-образы -> расшифровка
     Пример: «Лето» -> «о ле то о» -> «круг летит то кругом» (солнце)
     Пример: «Блеск» -> «бо о ле уз око» -> «живой круг летит остро глаз»
     Пример: «Абориген» -> «о берег он» -> «кругом берега он»
   — НМУ ВКС уже набрал 168 000 слов вручную
   — Задача: автоматизировать процедуру разбора и реконструкции
   — Импорт существующей базы 168 000 слов
   — ВК-генератор языка (ВК-О процессор): автоматический разбор
     нового слова на слоги-образы по правилам ВК-алфавита
   — Поиск слов и картинок в интернете для привязки образов

Блок 5. Текстовые алгоритмы действий
   — сложение, вычитание, умножение, деление
   — выполнение последовательности шагов по тексту на русском

Блок 6. Текст -> диафильм, диафильм -> текст
   — набор картинок из текста
   — текст из набора картинок

Блок 7. Решение технических задач по тексту
   — интерпретация алгоритма, записанного текстом
   — пример: геометрическая задача

Блок 8. Мобильное приложение для пассажира
   — адаптация модулей под телефон

Блок 9. Приложение для водителя
   — интерфейс водителя: маршрут, остановки, статус салона
   — связь с системой подсчёта пассажиров (Программа 2)
   — уведомления и отчёты для водителя

ПОДРОБНО О БЛОКЕ 4 (ВК-СЛОВАРЬ)
══════════════════════════════════════════════════════════════

Что такое ВК-словарь:
Реконструкционный словарь русского языка, в котором каждое
слово раскладывается на древние слоги-образы.

Формат записи (из существующей базы НМУ ВКС):

  Слово: Абориген
  Разбор 1: о берег он
  Значение 1: Кругом берега он
  Разбор 2: о бо ора го он
  Значение 2: Кругом живет делает путь он (местный)

  Слово: Абразив
  Разбор 1: обо рез бо
  Значение 1: Вода режет быть
  Разбор 2: о бо рез бо
  Значение 2: Кругом быть резать живо

  Слово: Лето
  Разбор: о ле то о
  Значение: Круг летит то кругом (солнце в заполярной области)

Что нужно сделать:
1. Импортировать существующую базу 168 000 слов в SQLite.
2. Создать ВК-генератор (ВК-О процессор): программу, которая
   автоматически разбивает новое слово на слоги-образы по
   правилам ВК-алфавита и подбирает значения.
3. Интерфейс для просмотра, поиска и редактирования словаря.
4. Экспорт словаря в удобные форматы.
