PROGRAM 2 - QUICK START

1) Unzip package.
2) Double-click run_program_2.bat
3) On the first start choose the camera source and press Start.
4) On the next starts the last camera source is loaded automatically.
5) To change camera, route, vehicle, door, or stop list later, run run_program_2_settings.bat.
6) During work you can press N or click SET on the panel to reopen settings and change source, counting, and performance parameters.

No Python installation is required.
Do not remove folder dist.
Reports are saved into folder reports near the program.
